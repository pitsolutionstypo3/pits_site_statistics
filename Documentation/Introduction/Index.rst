What does it do?
================

Pits Site Statistics simply retrieves the google analytics data through google analytics API and based on data recieved it plots graphical view of the data.

To know more about activating analytic api and creating credentials you can refer Google's document for analytics API here:: https://developers.google.com/analytics/devguides/reporting/core/v3/quickstart/service-php
